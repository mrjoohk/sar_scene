# Requirements Baseline: BL-TEST-001

## REQ-SAR-IMG-001
- **Statement**: The SAR imaging system SHALL meet the specified performance targets for cross-range resolution and processing latency.
- **Type**: performance
- **Priority**: must
- **Verification**: test
- **Acceptance**: Cross-range resolution shall be <= 0.3 m for the defined scene; end-to-end processing latency shall be <= 2.0 s per image under nominal load.
